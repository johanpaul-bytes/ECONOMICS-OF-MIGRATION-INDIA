# Economics of Migration: Brain Drain vs Brain Gain (India)

This repository contains an undergraduate research paper examining the economic
impact of skilled migration from India, focusing on brain drain and brain gain.

## Research Questions
- What drives skilled migration from India?
- How does brain drain affect key sectors such as IT, healthcare, and academia?
- Can remittances and reverse migration offset these losses?

## Data Sources
- World Bank Migration & Remittances Database
- RBI Remittance Reports
- NSS Migration in India (2007â€“08)
- Ministry of External Affairs (Indian Diaspora Data)

## Methodology
Qualitative analysis using secondary data, sector-wise analysis, and regional comparisons.

## Key Findings
- Skilled migration has increased sharply post-1990s globalization
- Remittances significantly support household income, especially in Kerala and Punjab
- Brain gain occurs through diaspora networks and reverse migration, but unevenly
